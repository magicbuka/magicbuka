### Руководитель направления Data Science

18+ лет управленческого опыта из них 9+ лет в IT:
- реализовываю проекты полного цикла по разработке систем с применением ИИ;
- координирую команды составом 10+ специалистов;
- умею презентовать результаты ТОП-менеджменту.

10+ проектов по анализу данных в промышленном секторе.

Умею:
- выстраивать эффективную коммуникацию с заказчиками (внутренними/внешними) и командой;
- просто и понятно излагать сложные концепции в части аналитических моделей и технической реализации;
- расставлять приоритеты и работать в команде в рамках жёстких дедлайнов.
